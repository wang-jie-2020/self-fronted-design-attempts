# 网站视觉风格调研

调研时间：2026-06-29

本文档总结当前常见、可落地的网站视觉风格。每种风格包含识别方式、视觉细节、适用场景和可访问参考 URL，便于后续做产品视觉方向选择。

## 快速结论

如果是 AI / SaaS / 开发者工具类产品，优先考虑：

- Bento Grid + Premium SaaS：信息清晰，可信度高，适合功能型产品。
- Dark Glass + Terminal Aesthetic：科技感强，适合 AI、数据、开发者工具。
- Editorial + 轻 3D：适合需要品牌叙事、作品展示或发布页的项目。

如果是品牌、活动页或年轻化产品，优先考虑：

- Neo-brutalism：辨识度高，适合反传统、创作者、活动页面。
- Dopamine Design：明亮、年轻、消费品导向。
- Retro / Y2K：适合音乐、社区、潮流、艺术项目。

| 风格 | 核心印象 | 适合场景 |
| --- | --- | --- |

| Bento Grid / 模块化卡片 | 信息密度高、结构清楚 | 多功能产品、AI 工具、产品官网 |
| Dark Glass / AI Command Center | 黑底、玻璃、光晕、控制台 | AI、DevTool、数据、安全、Web3 |

## 1. Premium SaaS / 极简产品感

### 视觉识别

- 大面积留白。
- 首屏通常由大标题、副标题、CTA 和产品截图组成。
- 使用细线、低饱和渐变、轻微阴影和精致动效。
- 视觉目标不是炫，而是显得可靠、现代、可扩展。

### 细节特征

- 布局：居中首屏、两栏介绍、功能分区清晰。
- 字体：现代无衬线，标题字重大，正文克制。
- 色彩：黑白灰为主，辅以低饱和蓝、青、紫、米色。
- 组件：圆角卡片、细边框、产品截图、浅层阴影。
- 动效：淡入、滚动 reveal、按钮 hover 微动效。

### 适合场景

- SaaS 官网。
- AI 工具。
- B2B 产品。
- 效率工具。
- 开发者产品的营销页。

### 参考 URL

- https://linear.app/
- https://stripe.com/
- https://vercel.com/

## 2. Bento Grid / 模块化卡片

### 视觉识别

- 页面由不等宽卡片组成。
- 每张卡片讲一个功能点或一个价值点。
- 常见组合：大卡片展示主功能，小卡片展示辅助能力。
- 信息多，但通过网格保持整洁。

### 细节特征

- 布局：2 到 4 列网格，不同卡片尺寸混排。
- 字体：短标题、短描述，避免长段落。
- 色彩：浅灰底 + 白卡，或深色底 + 半透明卡。
- 组件：图标、mini chart、产品 UI 截图、代码片段、状态标签。
- 动效：卡片分批入场，hover 时轻微上浮或加亮。

### 适合场景

- 功能较多的 SaaS。
- AI 产品。
- 工具集合。
- 个人作品集中的项目展示。
- 首页中段的功能说明。

### 参考 URL

- https://www.raycast.com/
- https://www.notion.com/
- https://www.framer.com/
- https://godly.website/?styles=%5B%22bento-grid%22%5D

## 3. Dark Glass / AI Command Center

### 视觉识别

- 深色背景。
- 半透明玻璃面板。
- 柔和光晕、渐变雾面、命令面板。
- 常出现类似控制台、数据面板或 AI 输入框的元素。

### 细节特征

- 布局：深色背景上叠加 dashboard、console、command palette。
- 字体：现代无衬线 + 少量等宽字体。
- 色彩：黑、深蓝、炭灰、墨绿，搭配青色、蓝紫、荧光绿光晕。
- 组件：毛玻璃卡片、半透明边框、代码窗口、状态灯、输入框。
- 动效：背景粒子、光晕移动、终端文字、卡片浮动。

### 适合场景

- AI 产品。
- 开发者工具。
- 数据分析平台。
- 安全产品。
- Web3 / infra 产品。

### 参考 URL

- https://spline.design/
- https://cursor.com/
- https://resend.com/
- https://www.dimension.dev/
- https://styles.refero.design/design-styles/dark-mode-websites

## 4. 3D / WebGL / 沉浸式

### 视觉识别

- 首屏通常有大型 3D 模型、空间场景或可交互物体。
- 页面有明显的景深、镜头运动和滚动叙事。
- 文案相对少，视觉本身承担主要表达。

### 细节特征

- 布局：3D 场景占据视觉中心，文字围绕或覆盖在其上。
- 字体：大标题、短文案，给 3D 留空间。
- 色彩：根据品牌决定，可科技黑、苹果式白、游戏高饱和。
- 组件：Canvas/WebGL、3D 模型、视差层、scroll-driven animation。
- 动效：鼠标跟随、滚动旋转、镜头推进、模型拆解。

### 适合场景

- 创意机构。
- 硬件产品。
- 设计工具。
- 高预算营销页。
- 需要强记忆点的品牌站。

### 参考 URL

- https://spline.design/
- https://www.apple.com/apple-vision-pro/
- https://lusion.co/
- https://land-book.com/?style=3D&view=desktop

## 5. Neo-brutalism / 新粗野主义

### 视觉识别

- 粗黑边框。
- 硬阴影。
- 高对比撞色。
- 看起来故意“不精致”，但非常有识别度。

### 细节特征

- 布局：块状、直接、不追求传统优雅网格。
- 字体：粗体、大字、display font，甚至怪异字体。
- 色彩：黄黑、粉黑、蓝红、白黑等强对比组合。
- 组件：3 到 5px 黑色描边、硬投影、贴纸、粗按钮、不规则图形。
- 动效：突兀、快速、弹跳，hover 位移明显。

### 适合场景

- 独立品牌。
- 创作者平台。
- 活动页。
- 年轻化产品。
- 需要反传统气质的网站。

### 参考 URL

- https://www.gumroad.com/
- https://www.mschf.com/
- https://brutalistwebsites.com/
- https://www.awwwards.com/awwwards/collections/brutalism/

## 6. Editorial / Magazine / 数字杂志感

### 视觉识别

- 强排版。
- 大标题。
- 大图和分栏。
- 页面像一篇可滚动的数字杂志或深度报道。

### 细节特征

- 布局：章节式滚动、图文穿插、分栏、引用块。
- 字体：Serif + Sans 组合常见，标题更有编辑部气质。
- 色彩：黑白为主，也常用米色、纸张色、照片主导色。
- 组件：大图、图表、注释、章节编号、互动可视化。
- 动效：scrollytelling，滚动时图表、地图、图片发生变化。

### 适合场景

- 品牌故事。
- 年度报告。
- 研究报告。
- 媒体专题。
- 个人作品集。

### 参考 URL

- https://pudding.cool/
- https://www.nytimes.com/interactive/
- https://www.bloomberg.com/graphics

## 7. Retro / Y2K / 怀旧互联网

### 视觉识别

- 像早期互联网、旧软件、音乐播放器、聊天室或桌面窗口。
- 有像素、扫描线、霓虹、marquee、老式 UI 的感觉。

### 细节特征

- 布局：旧网页、桌面窗口、播放器、卡片堆叠。
- 字体：像素字体、系统字体、复古 serif、压缩字体。
- 色彩：霓虹、海滩色、紫蓝渐变、米黄、CRT 绿色。
- 组件：仿 Windows 窗口、老式按钮、贴纸、CD、磁带、播放器。
- 动效：闪烁、光标特效、加载动画、滚动字幕。

### 适合场景

- 音乐项目。
- 社区产品。
- 潮流品牌。
- 艺术项目。
- 个人站。

### 参考 URL

- https://poolsuite.net/
- https://spacehey.com/
- https://www.are.na/

## 8. Dopamine / 高饱和快乐色

### 视觉识别

- 高饱和颜色。
- 圆润图形。
- 贴纸感、气泡感、玩具感。
- 情绪非常外放。

### 细节特征

- 布局：松散、圆润，像包装设计或社媒视觉。
- 字体：圆体、胖字体、手写字体、display font。
- 色彩：粉、黄、橙、蓝、酸绿，大面积高饱和撞色。
- 组件：大圆角、气泡、贴纸、插画、表情符号式图形。
- 动效：弹性、跳跃、拖拽、旋转贴纸。

### 适合场景

- 美妆。
- 食品饮料。
- DTC 品牌。
- 年轻用户产品。
- 快闪活动页。

### 参考 URL

- https://www.starface.world/
- https://www.oatly.com/
- https://www.lush.com/

## 9. Luxury Minimalism / 奢侈极简

### 视觉识别

- 大图。
- 很少文字。
- 节奏慢。
- 低饱和色和精致图片。
- 页面看起来不急着说服用户。

### 细节特征

- 布局：大图优先，导航轻，信息密度低。
- 字体：高级 serif、细字重 sans、字距拉开。
- 色彩：黑、白、米色、棕色、灰绿，饱和度低。
- 组件：少量按钮、轻导航、产品图、故事图。
- 动效：慢速淡入、平滑滚动、图片遮罩 reveal。

### 适合场景

- 奢侈品。
- 香氛。
- 家居。
- 建筑。
- 摄影。
- 艺术品牌。

### 参考 URL

- https://www.aesop.com/
- https://www.jacquemus.com/
- https://www.bang-olufsen.com/

## 10. Developer Docs / Terminal Aesthetic

### 视觉识别

- 代码块、终端窗口、API 参数表。
- 页面重点是可读性、结构和查找效率。
- 视觉风格服务于开发者信任感。

### 细节特征

- 布局：左侧导航 + 主内容 + 右侧目录，或首屏直接展示代码。
- 字体：Sans + Monospace。
- 色彩：深色或浅色均可，常见黑底代码块、绿色/蓝色状态色。
- 组件：代码块、复制按钮、参数表、终端窗口、版本标签。
- 动效：较少，重点是速度、清晰、可读。

### 适合场景

- API 文档。
- SDK 文档。
- 开源项目。
- 开发者工具。
- 技术产品官网。

### 参考 URL

- https://docs.astro.build/
- https://supabase.com/docs
- https://docs.stripe.com/

## 设计选型建议

| 项目类型 | 推荐方向 |
| --- | --- |
| AI / SaaS 工具 | Bento Grid + Dark Glass |
| 开发者产品 | Premium SaaS + Terminal Aesthetic |
| 个人作品集 | Editorial + 轻 3D |
| 创作者 / 社区产品 | Neo-brutalism + Retro |
| 消费品牌 | Dopamine 或 Luxury Minimalism |
| 活动页 / 营销页 | 3D / WebGL 或 Neo-brutalism |

## 落地注意事项

- 不要只复制表面元素。先确定产品气质，再选择对应风格。
- Bento Grid 容易变成普通卡片堆砌，需要用卡片尺寸、信息优先级和视觉焦点做层次。
- Dark Glass 容易牺牲可读性，需要控制对比度和透明度。
- 3D / WebGL 会增加性能和开发成本，适合首屏或关键模块，不适合全站滥用。
- Neo-brutalism 看似简单，但排版、间距和颜色控制不好会显得廉价。
- Luxury Minimalism 高度依赖图片质量，素材不够好时不要强行使用。
- Developer Docs 风格首要目标是清晰和查找效率，装饰应让位于可读性。
